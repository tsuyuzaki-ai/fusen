# fusen - 付箋アプリ

シンプルなタスク管理アプリケーションです。ブラウザのLocalStorageを使用してデータを保存します。

## 概要

fusenは、シンプルで使いやすい付箋（タスク管理）アプリです。タスクの追加・編集・削除、保留機能、ドラッグ＆ドロップによる順序変更など、基本的なタスク管理機能を提供します。

## 主な機能

- **タスクの追加**: 新しいタスクを簡単に追加
- **タスクの編集**: クリックでタスク内容を編集可能
- **タスクの削除**: 不要なタスクを削除
- **タスクの並び替え**: ドラッグ＆ドロップでタスクの順序を自由に変更できます
- **保留機能**: タスクを保留状態に設定（保留中のタスクは下部に表示）
- **削除の取り消し**: Ctrl+Z（Mac: Cmd+Z）で最後に削除したタスクを復元できます
- **データの永続化**: ブラウザのLocalStorageに自動保存

## 技術スタック

- **HTML5**
- **CSS3**
- **JavaScript (jQuery 3.6.0)**
- **LocalStorage**（データ保存）

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/tsuyuzaki-ai/fusen.git
cd fusen
```

2. Webサーバーで公開
このプロジェクトは静的サイトのため、任意のWebサーバーで公開できます。

ローカルで確認する場合：
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合（http-serverをインストール済み）
npx http-server
```

ブラウザで `http://localhost:8000` にアクセスしてください。

## 使い方

1. **タスクの追加**: 上部の入力欄にタスクを入力し、「追加」ボタンをクリック、またはEnterキーを押します
2. **タスクの編集**: 各タスクの編集アイコン（ペン）をクリックして編集モードに入ります
3. **タスクの並び替え**: タスクをドラッグ＆ドロップして順序を自由に変更できます
4. **タスクの保留**: 「保留」ボタンをクリックすると、タスクが保留状態になります（保留中のタスクは自動的に下部に移動します）
5. **タスクの削除**: 「削除」ボタンをクリックしてタスクを削除します
6. **削除の取り消し**: 誤って削除してしまった場合、Ctrl+Z（Mac: Cmd+Z）を押すと最後に削除したタスクを復元できます

## ディレクトリ構成

```
fusen/
├── img/           # 画像ファイル
│   ├── note.png   # ファビコン
│   └── pen.png    # 編集アイコン
├── index.html     # メインページ
├── main.js        # メインのJavaScriptファイル
├── note.js        # 補助的なJavaScriptファイル
└── style.css      # スタイルシート
```

## データ保存

タスクデータはブラウザのLocalStorageに保存されます。ブラウザを閉じてもデータは保持されますが、ブラウザのデータをクリアすると削除されます。
